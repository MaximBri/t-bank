## Запуск фронтенда

Требования:
- Node.js 20+
- npm 10+

Установка Node.js и npm:
- Скачайте Node.js (LTS) с https://nodejs.org
- npm устанавливается вместе с Node.js

Установка зависимостей:
```bash
npm install
```

Запуск в dev-режиме:
```bash
npm run dev
```

Сборка:
```bash
npm run build
```

Запуск тестов:
```bash
npm test
```

Линт:
```bash
npm run lint
```
