# UI Test Case

## Test Case ID
`TC-SMOKE-UI-13`

## Title
Открытие раздела события "Участники"

## Feature / Module
UI / Events / Participants Section

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.
3. Существует доступное пользователю событие.

## Steps
1. Перейти на страницу `/events/{eventId}?section=participants`, где `{eventId}` это идентификатор существующего события.
2. Дождаться полной загрузки страницы.
3. Проверить, что раздел "Участники" открыт и отображается корректно.

## Expected Result (Ожидаемый результат)
Раздел события "Участники" открывается и отображается корректно и согласно макету.
