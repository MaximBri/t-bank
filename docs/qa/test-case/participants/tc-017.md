# Test Case

## Test Case ID
`TC-017`

## Title
Выход из события при отсутствии незакрытых задолженностей

## Feature / Module
Participants / `GET /events/{eventId}/leave/check`, `POST /events/{eventId}/leave`

## Preconditions
1. Пользователь `qa_member_01` авторизован.
2. Пользователь `qa_member_01` является участником события.
3. У пользователя `qa_member_01` нет незакрытых задолженностей и расходов, которые мешают выходу из события.

## Steps
1. Открыть нужное событие на главной странице.
2. Нажать кнопку Покинуть событие.
3. Дождаться открытия модального окна подтверждения выхода.
4. Нажать кнопку Покинуть в модальном окне.

## Expected Result (Ожидаемый результат)
Открывается модальное окно подтверждения выхода из события.
После подтверждения пользователь успешно покидает событие.
Отображается уведомление об успешном выходе из события.
Пользователь возвращается на главную страницу.
Ошибки 5XX отсутствуют.

## Postconditions
- Пользователь `qa_member_01` больше не состоит в выбранном событии.
