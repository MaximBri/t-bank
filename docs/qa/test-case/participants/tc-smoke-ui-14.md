# UI Test Case

## Test Case ID
`TC-SMOKE-UI-14`

## Title
Открытие формы приглашения участников в событие

## Feature / Module
UI / Participants / Invite Participants Form

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.
3. Существует доступное пользователю событие.

## Steps
1. Перейти на страницу `/events/{eventId}?section=participants`, где `{eventId}` это идентификатор существующего события.
2. Нажать кнопку открытия формы приглашения участников.
3. Проверить наличие основных элементов формы.

## Expected Result (Ожидаемый результат)
Форма приглашения участников в событие открывается и отображается корректно и согласно макету.
