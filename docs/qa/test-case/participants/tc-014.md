# Test Case

## Test Case ID
`TC-014`

## Title
Исключение пользователя из события владельцем

## Feature / Module
Participants / `DELETE /events/{eventId}/participants/{userId}`

## Preconditions
1. Пользователь `qa_owner_01` авторизован.
2. Пользователь `qa_owner_01` является владельцем события.
3. В событии есть минимум один участник, которого можно исключить.
4. Пользователь находится на главной странице приложения.

## Steps
1. Открыть нужное событие на главной странице.
2. Перейти во вкладку Участники.
3. Найти в списке участника со статусом Участник.
4. Нажать кнопку исключения участника.
5. Дождаться завершения операции.

## Expected Result (Ожидаемый результат)
Пользователь успешно исключен из события.
Отображается уведомление об успешном исключении пользователя.
Исключенный пользователь больше не отображается в списке участников события.
Ошибки 5XX отсутствуют.
