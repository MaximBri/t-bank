# UI Test Case

## Test Case ID
`TC-SMOKE-UI-07`

## Title
Открытие модального окна выхода из события

## Feature / Module
UI / Participants / Leave Event Modal

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.
3. Существует доступное пользователю событие.

## Steps
1. Перейти на страницу `/events/{eventId}`, где `{eventId}` это идентификатор существующего события.
2. Нажать кнопку выхода из события.
3. Дождаться открытия модального окна.
4. Проверить наличие основных элементов модального окна выхода из события.

## Expected Result (Ожидаемый результат)
Модальное окно выхода из события открывается и отображается корректно и согласно макету.
