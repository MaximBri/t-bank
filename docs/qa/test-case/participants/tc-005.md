# Test Case

## Test Case ID
`TC-005`

## Title
Присоединение второго пользователя по пригласительной ссылке

## Feature / Module
Participants / `POST /groups/join`

## Preconditions
1. Пользователь `qa_owner_01` создал событие и получил пригласительную ссылку.
2. В системе существует второй пользователь `qa_member_01`.
3. Пользователь `qa_member_01` авторизован.
4. Используется валидная пригласительная ссылка события.

## Steps
1. Открыть пригласительную ссылку под пользователем `qa_member_01`.
2. Выполнить действие присоединения к событию.
3. Дождаться завершения операции.

## Expected Result (Ожидаемый результат)
Заявка пользователя успешно создана.
Открыт экран Страница Ивента.
Oшибки 5XX отсутствуют.
