# UI Test Case

## Test Case ID
`TC-SMOKE-UI-12`

## Title
Открытие раздела события "Взаиморасчеты"

## Feature / Module
UI / Events / Settlements Section

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.
3. Существует доступное пользователю событие.

## Steps
1. Перейти на страницу `/events/{eventId}?section=settlements`, где `{eventId}` это идентификатор существующего события.
2. Дождаться полной загрузки страницы.
3. Проверить, что раздел "Взаиморасчеты" открыт и отображается корректно.

## Expected Result (Ожидаемый результат)
Раздел события "Взаиморасчеты" открывается и отображается корректно и согласно макету.
