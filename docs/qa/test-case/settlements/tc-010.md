# Test Case

## Test Case ID
`TC-010`

## Title
Подтверждение получения средств по расчету

## Feature / Module
Settlements / `POST /settlements/{settlementId}/confirm`

## Preconditions
1. В событии есть расчет в статусе ожидания подтверждения.
2. Авторизован пользователь, который должен подтвердить получение средств.

## Steps
1. Открыть раздел взаиморасчетов.
2. Найти расчет в статусе ожидания подтверждения.
3. Выполнить действие подтверждения получения средств.

## Expected Result (Ожидаемый результат)
Расчет успешно подтвержден.
Отображается уведомление о подтверждении.
Ошибки 5XX отсутствуют.

## Postconditions
- Выбранный расчет завершен.
