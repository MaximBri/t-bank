`TC-011`

## Title

## Feature / Module

## Preconditions
1. Пользователь `qa_owner_01` авторизован.
2. Пользователь находится на главной странице приложения.

3. Нажать кнопку Выйти.

Открывается страница авторизации.
Отображается уведомление об успешном выходе из аккаунта.

- Активная сессия пользователя завершена.
