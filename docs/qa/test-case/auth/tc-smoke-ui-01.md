# UI Test Case

## Test Case ID
`TC-SMOKE-UI-01`

## Title
Открытие страницы логина без авторизации

## Feature / Module
UI / Authentication / Login Page

## Preconditions
1. Frontend запущен и доступен
2. Пользователь не авторизован в приложении.

## Steps
1. Перейти на страницу `/login`.
2. Дождаться полной загрузки страницы.
3. Проверить наличие формы авторизации на экране.

## Expected Result (Ожидаемый результат)
Страница логина открывается без авторизации и отображается корректно и согласно макету.

Форма авторизации доступна пользователю.
