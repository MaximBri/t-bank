# Test Case

## Test Case ID
`TC-001`

## Title
Регистрация нового пользователя

## Feature / Module
Authentication / `POST /auth/register`

## Preconditions
1. Пользователь с логином `qa_new_user_01` не существует.
2. Для регистрации используются данные: username `qa_new_user_01`, password `QaPass123!`.

## Steps
1. Открыть экран регистрации.
2. Заполнить логин.
3. Заполнить пароль.
4. Повторить пароль.
5. Нажать кнопку регистрации.

## Expected Result (Ожидаемый результат)
Новый пользователь успешно зарегистрирован. Создана активная сессия пользователя. Ошибки валидации и 5XX отсутствуют.

## Postconditions
- В системе создан новый пользователь `qa_new_user_01`.
