# UI Test Case

## Test Case ID
`TC-SMOKE-UI-02`

## Title
Открытие страницы регистрации без авторизации

## Feature / Module
UI / Authentication / Register Page

## Preconditions
1. Frontend запущен и доступен
2. Пользователь не авторизован в приложении.

## Steps
1. Перейти на страницу `/register`.
2. Дождаться полной загрузки страницы.
3. Проверить наличие формы регистрации на экране.

## Expected Result (Ожидаемый результат)
Страница регистрации открывается без авторизации и отображается корректно и согласно макету.

Форма регистрации доступна пользователю.
