# Test Case

## Test Case ID
`TC-013`

## Title
Обновление данных профиля в личном кабинете

## Feature / Module
User Profile / `PATCH /me`

## Preconditions
1. Пользователь `qa_owner_01` авторизован.
2. Пользователь находится на главной странице приложения.
3. Для обновления профиля используются данные: имя `Иван QA`, фамилия `Петров QA`, email `qa.updated.profile@example.com`.

## Steps
1. Открыть меню профиля на главной странице.
2. Перейти в Личный кабинет.
3. Убедиться, что открыта вкладка Профиль.
4. Нажать кнопку Редактировать профиль.
5. Изменить имя.
6. Изменить фамилию.
7. Изменить email.
8. Нажать кнопку Сохранить.

## Expected Result (Ожидаемый результат)
Данные профиля успешно обновлены.
Отображается уведомление об успешном обновлении данных профиля.
На странице профиля отображаются новые имя, фамилия и email.
Ошибки валидации и 5XX отсутствуют.

## Postconditions
- Для пользователя `qa_owner_01` сохранены обновленные данные профиля.
