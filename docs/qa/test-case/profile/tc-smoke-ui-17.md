# UI Test Case

## Test Case ID
`TC-SMOKE-UI-17`

## Title
Открытие раздела личного кабинета "Профиль"

## Feature / Module
UI / User Profile / Profile Section

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.

## Steps
1. Перейти на страницу `/profile?section=profile`.
2. Дождаться полной загрузки страницы.
3. Проверить наличие основных элементов раздела "Профиль".

## Expected Result (Ожидаемый результат)
Раздел личного кабинета "Профиль" открывается и отображается корректно и согласно макету.
