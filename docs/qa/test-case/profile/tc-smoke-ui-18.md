# UI Test Case

## Test Case ID
`TC-SMOKE-UI-18`

## Title
Открытие раздела личного кабинета "Уведомления"

## Feature / Module
UI / User Profile / Notifications Section

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.

## Steps
1. Перейти на страницу `/profile?section=notifications`.
2. Дождаться полной загрузки страницы.
3. Проверить наличие основных элементов раздела "Уведомления".

## Expected Result (Ожидаемый результат)
Раздел личного кабинета "Уведомления" открывается и отображается корректно и согласно макету.
