`TC-SMOKE-UI-16`

## Title
Открытие страницы личного кабинета

## Feature / Module
UI / User Profile / Personal Account

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.

1. Перейти на страницу `/profile`.
2. Дождаться полной загрузки страницы.
3. Проверить наличие основных разделов личного кабинета.

## Expected Result (Ожидаемый результат)
Страница личного кабинета открывается и отображается корректно и согласно макету.
