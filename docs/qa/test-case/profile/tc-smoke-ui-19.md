## Title

## Feature / Module
UI / User Profile / Security Section

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.

## Steps
1. Перейти на страницу `/profile?section=security`.
2. Дождаться полной загрузки страницы.
3. Проверить наличие основных элементов раздела "Безопасность".

## Expected Result (Ожидаемый результат)
Раздел личного кабинета "Безопасность" открывается и отображается корректно и согласно макету.
