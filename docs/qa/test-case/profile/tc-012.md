# Test Case

## Test Case ID
`TC-012`

## Title
Изменение пароля в личном кабинете

## Feature / Module
User Profile / `POST /me/password`

## Preconditions
1. Пользователь `qa_owner_01` авторизован.
2. Пользователь находится на главной странице приложения.
3. Для смены пароля используются данные: текущий пароль `QaPass123!`, новый пароль `QaNewPass123!`, подтверждение нового пароля `QaNewPass123!`.

## Steps
1. Открыть меню профиля на главной странице.
2. Перейти в Личный кабинет.
3. Перейти во вкладку Безопасность.
4. Заполнить поле Текущий пароль.
5. Заполнить поле Новый пароль.
6. Заполнить поле Подтвердите новый пароль.
7. Нажать кнопку Изменить пароль.

## Expected Result (Ожидаемый результат)
Пароль пользователя успешно изменен.
Отображается уведомление об успешной смене пароля.
Ошибки валидации и 5XX отсутствуют.

## Postconditions
- Для пользователя `qa_owner_01` установлен новый пароль `QaNewPass123!`.
