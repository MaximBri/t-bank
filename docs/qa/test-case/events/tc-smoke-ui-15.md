# UI Test Case

## Test Case ID
`TC-SMOKE-UI-15`

## Title
Открытие экрана истории действий

## Feature / Module
UI / Events / History Page

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.
3. Существует доступное пользователю событие.

## Steps
1. Перейти на страницу `/events/{eventId}`, где `{eventId}` это идентификатор существующего события.
2. Открыть экран истории действий.
3. Дождаться полной загрузки страницы или блока истории.
4. Проверить, что экран истории действий отображается корректно.

## Expected Result (Ожидаемый результат)
Экран истории действий открывается и отображается корректно и согласно макету.
