# UI Test Case

## Test Case ID
`TC-SMOKE-UI-05`

## Title
Открытие экрана создания события

## Feature / Module
UI / Events / Create Event

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.

## Steps
1. Перейти на страницу `/`.
2. Нажать кнопку открытия формы создания события.
3. Дождаться открытия модального окна.
4. Проверить наличие основных элементов формы создания события.

## Expected Result (Ожидаемый результат)
Экран создания события открывается и отображается корректно и согласно макету.
