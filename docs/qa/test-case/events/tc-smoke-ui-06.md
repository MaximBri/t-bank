# UI Test Case

## Test Case ID
`TC-SMOKE-UI-06`

## Title
Открытие страницы события

## Feature / Module
UI / Events / Event Page

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.
3. Существует доступное пользователю событие.

## Steps
1. Перейти на страницу `/events/{eventId}`, где `{eventId}` это идентификатор существующего события.
2. Дождаться полной загрузки страницы.
3. Проверить отсутствие критических ошибок отображения.
4. Проверить наличие основных полей формы.

## Expected Result (Ожидаемый результат)
Страница события открывается и отображается корректно и согласно макету.
