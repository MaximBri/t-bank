# Test Case

## Test Case ID
`TC-004`

## Title
Получение пригласительной ссылки для события

## Feature / Module
Invitations / `GET /events/{eventId}/invite-link`

## Preconditions
1. Пользователь `qa_owner_01` авторизован.
2. У пользователя есть созданное событие.

## Steps
1. Открыть карточку события владельца.
2. Перейти в раздел Участники.
3. Запросить текущую пригласительную ссылку нажав на кнопку Пригласить участников.

## Expected Result (Ожидаемый результат)
Для события отображается валидная пригласительная ссылка.
Ссылка доступна для копирования и дальнейшего использования.
Ошибки 5XX отсутствуют.
