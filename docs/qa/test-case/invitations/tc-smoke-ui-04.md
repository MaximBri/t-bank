# UI Test Case

## Test Case ID
`TC-SMOKE-UI-04`

## Title
Просмотр модального окна принятия приглашения в событие

## Feature / Module
UI / Invitations / Accept Invitation Modal

## Preconditions
1. Frontend запущен и доступен
2. Пользователь авторизован в приложении.
3. У пользователя есть доступное приглашение в событие - ссылка.

## Steps
1. Перейти по пригласительной ссылке.
2. Дождаться открытия модального окна.
3. Проверить наличие основных элементов модального окна принятия приглашения.

## Expected Result (Ожидаемый результат)
Модальное окно принятия приглашения в событие открывается и отображается корректно и согласно макету.
