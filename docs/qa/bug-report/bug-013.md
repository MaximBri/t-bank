## Bug ID
`BUG-013`

## Title
Редактирование события возвращает `500 Internal Server Error`

## Severity (Серьёзность дефекта)
- [x] Высокая
- [ ] Средняя
- [ ] Низкая

## Description
При попытке отредактировать существующее событие backend возвращает `500 Internal Server Error`.

В теле ответа возвращается:

```json
{"message":"An unexpected error occurred","status":500}
```

## Preconditions
1. Frontend или API запущены локально.
2. Backend запущен локально.
3. Пользователь авторизован.
4. Существует событие, где текущий пользователь является владельцем.

## Steps to Reproduce
1. Открыть существующее событие.
2. Изменить данные события.
3. Отправить запрос на сохранение изменений.

## Expected Result (Ожидаемый результат)
Событие успешно обновляется, backend возвращает успешный статус.

## Actual Result (Фактический результат)
Backend возвращает `500 Internal Server Error`.

```json
{"message":"An unexpected error occurred","status":500}
```

## Environment
- Стенд: local
- Backend API

## Attachments
- Логи / ответ API с `500`
