## Bug ID
`BUG-008`

## Title
CORS-ошибка при регистрации 

## Severity (Серьёзность дефекта)
- [x] Высокая
- [ ] Средняя
- [ ] Низкая

## Description
При попытке выполнить регистрацию через frontend пользователь не может успешно войти в приложение.

В локальной конфигурации frontend браузер получает CORS-ошибку (`Invalid CORS request`).


## Preconditions
1. Frontend запущен локально.
2. Backend запущен локально.
3. Frontend настроен на backend `http://localhost:8081`

## Steps to Reproduce
1. Открыть страницу регистрации во frontend `http://localhost:5173/register`.
2. Ввести валидные данные пользователя.
3. Нажать кнопку регистрации.

## Expected Result (Ожидаемый результат)
Пользователь успешно регистрируется.

## Actual Result (Фактический результат)
Пользователь не может зарегистрироваться в приложении.

Браузер получает CORS-ошибку при обращении frontend к backend `Не удалось зарегистрироваться`.
В DevTools / Network воспроизводится `403 Forbidden`

```text
Invalid CORS request
```


## Environment
- Браузер (Chrome, стабильная версия `147.0.7727.55`, Linux/Pop!_OS 24.04 LTS x86_64)
- Стенд: local
