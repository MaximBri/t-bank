## Bug ID
`BUG-023`

## Title
На странице события кнопка `Завершить событие` не работает

## Severity (Серьёзность дефекта)
- [x] Высокая
- [ ] Средняя
- [ ] Низкая

## Description
На странице события для владельца отображается кнопка `Завершить событие`, однако при нажатии сценарий завершения события не выполняется.

Наблюдаемое поведение:
- request на backend не отправляется;
- статус события не изменяется.

## Preconditions
1. Frontend запущен локально.
2. Backend запущен локально.
3. Пользователь авторизован как владелец события.

## Steps to Reproduce
1. Открыть страницу события от лица владельца.
2. Нажать кнопку `Завершить событие`.
3. Проверить network-запросы и состояние страницы.

## Expected Result (Ожидаемый результат)
После нажатия кнопки отправляется request на завершение события.
Статус события изменяется на `Завершено`.
Пользователь видит обновленное состояние события.

## Actual Result (Фактический результат)
Request на backend не отправляется.
Статус события не меняется.
Кнопка не выполняет ожидаемое действие.

- Стенд: local
