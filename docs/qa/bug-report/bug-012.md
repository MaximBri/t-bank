## Bug ID
`BUG-012`

## Title
Редактирование расхода возвращает `500 Internal Server Error`

## Severity (Серьёзность дефекта)
- [x] Высокая
- [ ] Средняя
- [ ] Низкая

## Description
При попытке отредактировать существующий расход backend возвращает `500 Internal Server Error`.

В теле ответа возвращается:

```json
{"message":"An unexpected error occurred","status":500}
```

## Preconditions
1. Frontend или API запущены локально.
2. Backend запущен локально.
3. Пользователь авторизован.
4. Существует событие.
5. В событии существует расход, доступный для редактирования.

## Steps to Reproduce
1. Открыть существующий расход.
2. Изменить данные расхода.
3. Отправить запрос на сохранение изменений.

## Expected Result (Ожидаемый результат)
Расход успешно обновляется, backend возвращает успешный статус.

## Actual Result (Фактический результат)
Backend возвращает `500 Internal Server Error`.

```json
{"message":"An unexpected error occurred","status":500}
```

## Environment
- Стенд: local
- Backend API

## Attachments
- Логи / ответ API с `500`
