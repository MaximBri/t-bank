## Bug ID
`BUG-019`

## Title
Участник не может присоединиться по пригласительной ссылке

## Severity (Серьёзность дефекта)
- [x] Высокая
- [ ] Средняя
- [ ] Низкая

## Description
Переход по пригласительной ссылки в модальном окне `Приглашение участников` не позволяет пройти сценарий приглашения участника.

Пример ссылки:

```text
http://localhost:5173/invite/10fa0264-8799-47aa-9c02-2e996bf49ff4/6h8Vu8VbXWNRjg2zaqNTMOqmoebvnFzGlP6jLJT1hCA
```

Из-за этого невозможно полноценно проверить:
- приглашение по ссылке
- подтверждение участия
- дальнейшие сценарии participants / invite flow

## Preconditions
1. Frontend запущен локально.
2. Backend запущен локально.
3. Пользователь авторизован.
4. Существует событие, для которого можно сгенерировать invite link.
5. Второй пользователь авторизован в приложении.

## Steps to Reproduce
1. Открыть событие.
2. Открыть вкладку `Приглашение участников`.
3. Скопировать сгенерированную ссылку приглашения.
4. Перейти по ссылке от лица второго пользователя

## Expected Result (Ожидаемый результат)
Ссылка открывает рабочую страницу приглашения, где можно продолжить сценарий присоединения к событию.

## Actual Result (Фактический результат)
Ссылка не позволяет пройти сценарий приглашения. Проверка invite flow блокируется. Открывается домашняя страница в ивентами.

- Стенд: local
