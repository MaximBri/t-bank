## Bug ID
`BUG-011`

## Title
После успешного добавления события его изображение не отображается в карточке события

## Severity (Серьёзность дефекта)
- [ ] Высокая
- [x] Средняя
- [ ] Низкая

## Description
После успешного создания события карточка события отображается на главной странице, но изображение события не загружается.

Вместо изображения в карточке виден сломанный placeholder / alt-text.

## Preconditions
1. Frontend запущен локально.
2. Backend запущен локально.
3. Пользователь авторизован.
4. Доступна форма создания события.

## Steps to Reproduce
1. Открыть главную страницу приложения.
2. Нажать `Создать событие`.
3. Заполнить обязательные поля.
4. Добавить изображение события.
5. Успешно создать событие.
6. Вернуться к карточке созданного события на главной странице.

## Expected Result (Ожидаемый результат)
После успешного создания события изображение корректно отображается в карточке события.

## Actual Result (Фактический результат)
После успешного создания события карточка события отображается, но изображение не загружается.

Вместо изображения отображается broken image placeholder / alt-text.

## Environment
- Браузер: Chrome
- Стенд: local
